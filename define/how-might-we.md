# How Might We (HMW) - Projeto Pequi (Define Phase)

## HMW Statements
Baseados em perguntas de ideação, entrevistas (Itanielly) e problema central. Focam em oportunidades user-centered.

### HMW do Problema
1. **How might we reduzir diagnóstico tardio identificando contatos de forma confiável, sem depender de memória manual ACS?**
2. **How might we garantir completude do acompanhamento anual obrigatório para contatos?**
3. **How might we mitigar estigma que impede busca por atendimento?**
4. **How might we melhorar diagnóstico em fases iniciais, superando confusão com sintomas comuns (câimbras, pele)?**

### HMW da Solução (E se... → HMW)
1. **How might we permitir que ACS cadastrem contatos digitalmente/anônimos via celular, sem expor identidade do paciente?**
2. **How might we automatizar triagem sintomas contextualizada para contatos, gerando score risco que priorize visitas ACS?**
3. **How might we fazer o sistema lembrar automaticamente follow-up anual, independente de agenda manual?**
4. **How might we criar painel centralizado para ACS ver contatos ativos, status triagem e alertas imediatos?**

### HMW de Insights Adicionais (Reunião/ SINAN)
5. **How might we facilitar adesão tratamento pós-diagnóstico, combatendo reclusão por efeitos colaterais (pigmentação)?** (ex: lembretes doses, supervisão remota).
6. **How might we simplificar registro SINAN, evitando duplicidades e atrasos manuais?**
7. **How might we envolver grupos autocuidado (rodas conversa) digitalmente para educação anti-estigma?**
8. **How might we priorizar contatos <15 anos e fontes familiares sem expor?**
