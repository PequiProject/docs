# User Journey Maps - Projeto Pequi (Define Phase)

## Jornada do ACS (Agente Comunitário de Saúde)
**Persona:** Maria, 42 anos, 100+ famílias, smartphone básico.

| Fase | Ações Atuais (Problema) | Emoções | Oportunidades Pequi |
|------|--------------------------|---------|---------------------|
| **Descoberta Paciente** | Anota manual caderno; memória falha. | Frustração. | Cadastro digital anônimo rápido (QR/código familiar). |
| **Mapeamento Contatos** | Lista contatos de memória/agenda; esquece. | Ansiedade por falhas. | Triagem inicial sintomas → score risco auto. |
| **Follow-up Anual** | Visitas aleatórias; boletim SINAN manual semestral. | Pressão resultados. | Alertas painel priorizado; lembretes auto WhatsApp. |
| **Tratamento Paciente** | Acompanha doses 28 dias (supervisionada); baixa manual. | Desistências por reclusão paciente. | Lembretes doses; supervisão remota opcional. |
| **Relatório** | Digitaliza só fim tratamento; duplicidades. | Trabalho extra. | Relatórios auto-gerados, SINAN-ready. |

**Touchpoints:** Celular → Painel → Mensagens neutras.

## Jornada do Contato/Paciente
**Persona:** João, 28 anos, família diagnosticado, estigma alto.

| Fase | Ações Atuais (Problema) | Emoções | Oportunidades Pequi |
|------|--------------------------|---------|---------------------|
| **Contato Identificado** | Não notificado; ignora. | Negação, medo rótulo. | Cadastro anônimo; sem nome doença. |
| **Triagem Sintomas** | Espera visita ACS; esconde sintomas. | Vergonha (confunde pele simples). | Auto-triagem via WhatsApp: \"Check-up familiar\". |
| **Acompanhamento Anual** | Falha comparecimento; finge sem sintomas. | Alívio curto prazo, risco longo. | Lembrete discreto anual; score envia ACS se risco. |
| **Tratamento (se diagnosticado)** | Doses casa (auto); reclusa por pigmentação. | Desistência, isolamento. | Lembretes adesão; suporte virtual anti-estigma. |
| **Alta** | Sem follow-up; dados atrasados SINAN. | Recaída possível. | Confirmação auto; grupo autocuidado digital. |

**Pain Points Comuns:** Estigma, manualidade, abandono.

**Insights SINAN:** Boletins manuais → Digitalizar journey para fluxo contínuo.

**Visualização (ASCII - ACS Journey):**
```
[Paciente Dx] → [Cadastro Contatos Anon] → [Triagem/Score] → [Visita Priorizada]
                    ↓ Anual
[ Lembrete Auto ] ← [Painel ACS] → [Atualiza Status]
```

**Próximo: Validar journeys com entrevistas reais; refinar em workshops."
